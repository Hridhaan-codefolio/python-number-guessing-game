# Number Guessing Game

A console-based Number Guessing Game built in Python.

# Features
Three difficulty levels:
Easy (1–50, 10 attempts)
Medium (1–100, 7 attempts)
Hard (1–200, 5 attempts)

Intelligent hints:
Very Close
Close
Way Too Far
Too Small
Too Big

Statistics System:
Total Wins
Total Rounds Played
Current Win Streak
Best Win Streak
Win Rate Percentage
Last 10 Game Results History

Save & Load System:
Automatically saves player statistics to a text file
Loads previous statistics when the game starts

Menu System:
Play Game
View Statistics
Reset Statistics
Exit
